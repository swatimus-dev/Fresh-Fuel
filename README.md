# Fresh-Fuel
A simple Nutrition Information Website built using HTML, CSS, and JavaScript. The website provides basic nutrition knowledge, healthy eating tips, an interactive quiz, and a visit counter with a clean and user-friendly interface. With few recipes and nutrition information 
The website includes multiple sections such as key nutrition information, quick tips for healthy eating, a small interactive quiz, and a visit counter. The project focuses on creating an informative and visually organized interface.
Displays important nutrition information
Provides healthy eating tips
Includes a small interactive quiz
Shows a visit counter
10 healty recipes and BMI calculator
Clean and simple user interface
